SANTOS COY LEGACY — IMAGE REPLACEMENT INVENTORY
================================================

All images currently load from Unsplash as placeholders. Replace each <img src="..."> with
a local file in /images/ (e.g., src="images/hero-brush.jpg") when the real artifact is in hand.
Every <img> already has detailed alt text describing exactly what should fill that slot.

NOTES
-----
- The .frame CSS automatically applies sepia tone, paper border, slight rotation —
  plain photographs will take on the museum aesthetic without any editing.
- Recommended size: ~1400px on the long edge for hero/figure shots, ~900px for gallery.
- All paths are RELATIVE — the project drops directly into handoffpack.com/labl/coy/.
- For Wikimedia-Commons-sourced public-domain images (San Jacinto, Hays portrait, the
  Alamo, etc.), you can hot-link or save locally and credit in figcaption.
